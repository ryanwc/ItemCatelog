Add Restaurant

Create a new restaurant and add it to the restaurant list.
Note that the restaurant's menu will start with all of the base menu items (if any) for its cuisine.
You can view and modify cuisines and their base menu items here.

Important notes about cuisines:

  • All existing cuisines are listed in the dropdown menu below.
  • If you want the restaurant to have a cuisine that doesn't exist in the database yet, select "Custom (enter in text box below)" from the dropdown and type your custom cuisine in the box. This will create a new cuisine in the database and assign it to the restaurant.
  • A custom cuisine will not have any base menu items, so you will need to populate the restaurant's menu yourself.
  • If the restaurant has no specific cuisine type, you can select "No Specific Cuisine" from the dropdown and the restaurant will not get any items on its menu.
