Add Restaurant

Create a new restaurant and add it to the restaurant list.
Note that the restaurant's menu will start with all of the base menu items (if any) for its cuisine.
You can view and modify cuisines here and base menu items here.

All existing cuisines are listed in the dropdown menu below.
If you want to create a new cuisine, select "Custom (enter in text box below)" from the dropdown and type your custom cuisine in the box.
A custom cuisine will not have any base menu items, so you will need to populate the restaurant's menu yourself.
