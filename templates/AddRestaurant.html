Info for New Restaurant

Important notes:
  • The restaurant's menu will start with all of the base menu items (if any) for its cuisine.
  • If the restaurant doesn't have a specific cuisine, select "No Specific Cuisine" from the dropdown. The restaurant will not start with any items on its menu.
  • If the restaurant has a specific cuisine but the cuisine is not listed in the dropdown, select "Custom (enter in text box below)" from the dropdown and type the new cuisine in the box. This will create a new cuisine in the database with no base menu items and assign it to the restaurant. You'll have to manually add base menu items for the cuisine and restaurant menu items for the restaurant.
